3. ApiDocÂ¶

These are the ApiDocs of the project.

As explained in the Overview and configuration section. The project is subdivided in 2 main modules:

  • simulator: defines a set of remotes that send commands through SAL to simulated (or real) instances of different CSCs

  • csc_sim: defines a set of controllers that simulate different CSCs. These controllers receive commands from the remotes defined in simulator
